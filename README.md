# Introduction

This library supports Feature-Oriented Programming in TypeScript through the use of Object Algebras. In addition, as the current version of TypeScript (2.6) does [not support](https://github.com/Microsoft/TypeScript/issues/1213) Type Constructor Polymorphism, this library also provides a workaround in order to support more convenient composition of multiple algebra definitions.
